-= Note =-
These drivers are for RobotC 2.00 and higher and are part of the
version 2.2 of the 3rd Party RobotC Drivers suite.

What's new in 2.1?
* Many fixes in the DIMU driver:
  * Fixed some bugs in the dividers
  * Added optional Low Pass Filtering when initialising the gyro
  * The driver now reads 2 or 6 bytes at once for faster data sampling
* Fixed an old bug in FLAC that somehow managed to survive for this long
* Fixed some comments in HTIRS2-driver

-= API Documentation =-
The complete API documentation can be found in the html folder.
Just point your browser at the index.html file.


-= Downloads and support =-
These drivers can also be downloaded from:
http://sourceforge.net/projects/rdpartyrobotcdr/
